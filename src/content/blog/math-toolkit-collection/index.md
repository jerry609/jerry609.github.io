## 工具卡片字段

每篇工具文章统一保留这些字段：

| 字段 | 内容 |
| --- | --- |
| 工具名 | 这个工具叫什么 |
| 类型 | 模型、算子、检查器、解析求解、迭代算法或证书 |
| 输入 | 调用这个工具需要什么 |
| 输出 | 工具返回什么 |
| 公式 | 最小核心公式 |
| 依赖工具 | 需要先掌握哪些工具 |
| 用途 | 它通常解决什么问题 |
